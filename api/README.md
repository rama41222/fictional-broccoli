# Cycling and Weather API

## Setup

1. Clone this repo and install all the dependancies

```bash
git clone git@github.com:rama41222/fictional-broccoli.git

cd fictional-broccoli/api

yarn
```

2. Run dfdf

```bash
yarn run build # Build the project

yarn run start # Run the built code

yarn run dev # Debugger

yarn run dev:server # Runs the server using nodemon

yarn run test #runs all the unit tests
```
